Suizide und Arbeitslose im Vergleich

Indexierter Verlauf · Startwert 100

Beide Datenreihen werden indexiert. Das bedeutet: Beide Linien starten beim gleichen Wert 100. So werden nicht die absoluten Zahlen verglichen, sondern die Entwicklung über die Jahre. Der Vergleich zeigt keine direkte Ursache, sondern nur, ob sich die Verläufe ähnlich oder unterschiedlich bewegen.

Datenquellen

Bundesamt für Statistik
World Bank Open Data

Die Suiziddaten stammen vom Schweizer Bundesamt für Statistik. Die Arbeitslosenzahlen werden von der World Bank Open Data API geliefert. Die absolute Anzahl Arbeitsloser wird aus der Erwerbsbevölkerung und der Arbeitslosenquote berechnet.
