# Suizid in der Schweiz – Daten & Kontext

## Projektbeschreibung
Eine datengetriebene Webseite, die Suizidstatistiken der Schweiz (1970–2024) dynamisch visualisiert und mit gesellschaftlichen Indikatoren wie der Arbeitslosenquote in Beziehung setzt.

## APIs
- **Suizid-Statistik API**: https://suizid.mezaciru.myhostpoint.ch/api/
- **World Bank API**: https://api.worldbank.org/v2/country/CHE/indicator/SL.UEM.TOTL.ZS?format=json

## Technologien
- HTML, CSS, JavaScript
- Fetch API (async/await)
- Responsive Design
